Protools Commands

Command Description
// Gives you the selection tool
/pos1, /posa Sets the point A of the selection to your location
/pos2, /posb Sets the point B of the selection to your location
/set Sets the selected area to a certain block type
/fill Fills an area with the selected block type
/walls Sets a wall around the selected area
/wireframe Outlines the selected region with the given block type
/cut Cuts the selected area (sets it to air)
/copy Copies the selected area
/paste Pastes the copied area
/undo Undoes the last action
/replace <block> <block> Replaces the given block type in the selected area with another block
